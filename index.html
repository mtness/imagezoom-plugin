DEFAULT SETTING

data-imagezoom="true"に指定するとマウスオーバーする画像を拡大表示します

<img src="sampleimage/1.jpg" data-imagezoom="true">

ANOTHER IMAGE

data-imagezoomに指定された画像を拡大表示します。高画質の同画像を表示する場合に便利です。

<img src="sampleimage/2.jpg" data-imagezoom="sampleimage/1.jpg">

ZOOMVIEW SIZE

data-zoomviewsizeに指定されたサイズの拡大ビューを表示します。

<img src="sampleimage/1.jpg" data-imagezoom="true" data-zoomviewsize="[300,500]">

ZOOMVIEW POSITION

data-zoomviewpositionに[left]か[right]を指定された方向に拡大ビューを表示します。デフォルトは[right]です。
また、data-zoomviewmarginに数値を指定することで拡大ビューと画像のマージンを設定できます。

<img src="sampleimage/1.jpg" data-imagezoom="true" data-zoomviewposition="left" data-zoomviewmargin="20">

ZOOMLEVEL

data-magnificationに数値を指定すると拡大率を変更できます。デフォルトの倍率は3です。

<img src="sampleimage/1.jpg" data-imagezoom="true" data-magnification="10">

DESIGN

data-zoomviewborderでズームビューにボーダーを指定できます。
data-cursorでカーソル指定ができます。
data-cursorcolorでカーソルカラーを指定できます。
data-opacityでカーソルの透明度を指定できます。

<img src="sampleimage/2.jpg" data-imagezoom="true" data-zoomviewborder="6px solid #000" data-cursor="none" data-cursorcolor="52,152,219" data-opacity="0.9">

jQuery CODE

jQueryでのオプション指定。data属性の指定が優先されます。

<script>
$('[data-imagezoom]').imageZoom({
    cursorcolor:'255,255,255', //カーソル背景色
    opacity:0.5, //カーソル透明度
    cursor:'crosshair', //カーソルタイプ
    zindex:2147483647, //カーソルレイヤー
    zoomviewsize:[500,500], //ズームビューサイズ
    zoomviewposition:'right', //ズームビューの位置指定[left or right]
    zoomviewmargin:10, //ズームビューと画像のマージン
    zoomviewborder:'1px solid #000', //ズームビューのボーダースタイル
    magnification:3 //拡大率
});
</script>
